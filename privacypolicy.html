Política de Privacidade

WisePrior — Última atualização: maio de 2026

Coleta de Dados

O WisePrior não coleta, transmite nem compartilha informações pessoais identificáveis. Todos os dados gerados pelo uso do aplicativo permanecem exclusivamente no seu dispositivo.

Nenhum dado pessoal é enviado a servidores externos.

Dados Armazenados

As seguintes informações podem ser armazenadas localmente no dispositivo pelo aplicativo:

  • Tarefas criadas pelo usuário (título, descrição, prioridade, status)
  • Datas e horários de vencimento ou lembrete definidos pelo usuário
  • Preferências de ordenação e visualização da lista de tarefas

Esses dados são salvos em um banco de dados local (Room/SQLite) e não são sincronizados com a nuvem. Desinstalar o aplicativo remove permanentemente todos os dados armazenados.

Notificações

O WisePrior pode enviar notificações locais para lembretes de tarefas configurados pelo próprio usuário. As notificações são geradas inteiramente no dispositivo, sem qualquer envio de dados para servidores remotos. O usuário pode gerenciar ou desativar as notificações a qualquer momento nas configurações do sistema Android.

Serviços de Terceiros

O WisePrior não utiliza nenhum SDK de publicidade, rastreamento, análise (analytics) ou monetização de terceiros. Não há integração com Firebase Analytics, Crashlytics, Google AdMob, Facebook SDK ou qualquer serviço similar.

As únicas bibliotecas utilizadas são componentes oficiais do ecossistema Android/Jetpack (Compose, Room, WorkManager, Hilt), que operam exclusivamente no dispositivo local.

Permissões Solicitadas

O aplicativo solicita as seguintes permissões do sistema Android:

  • POST_NOTIFICATIONS Exibir notificações de lembrete de tarefas (obrigatório no Android 13 ou superior).
  • SCHEDULE_EXACT_ALARM Agendar alarmes precisos para que os lembretes disparem no horário exato definido pelo usuário.
  • RECEIVE_BOOT_COMPLETED Reagendar os lembretes automaticamente após o dispositivo ser reiniciado.
  • VIBRATE Vibrar o dispositivo quando uma notificação de lembrete for exibida.

Privacy Policy

WisePrior — Last updated: May 2026

Data Collection

WisePrior does not collect, transmit, or share any personally identifiable information. All data generated through use of the app remains exclusively on your device.

No personal data is sent to external servers.

Stored Data

The following information may be stored locally on the device by the app:

  • Tasks created by the user (title, description, priority, status)
  • Due dates and reminder times set by the user
  • Task list sorting and display preferences

This data is saved in a local database (Room/SQLite) and is not synced to the cloud. Uninstalling the app permanently removes all stored data.

Notifications

WisePrior may send local notifications for task reminders configured by the user. Notifications are generated entirely on the device, with no data sent to remote servers. Users can manage or disable notifications at any time through Android system settings.

Third-Party Services

WisePrior does not use any third-party advertising, tracking, analytics, or monetization SDKs. There is no integration with Firebase Analytics, Crashlytics, Google AdMob, Facebook SDK, or any similar service.

The only libraries used are official Android/Jetpack ecosystem components (Compose, Room, WorkManager, Hilt), which operate exclusively on the local device.

Requested Permissions

The app requests the following Android system permissions:

  • POST_NOTIFICATIONS Display task reminder notifications (required on Android 13 and above).
  • SCHEDULE_EXACT_ALARM Schedule precise alarms so reminders fire at the exact time set by the user.
  • RECEIVE_BOOT_COMPLETED Automatically reschedule reminders after the device reboots.
  • VIBRATE Vibrate the device when a reminder notification is displayed.
